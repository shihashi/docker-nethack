# docker-nethack
Dockerfile for Nethack

## How to play the game

```
$ git clone https://github.com/shihashi/docker-nethack
$ cd docker-nethack
$ ./nethack
```
Then the log files and the save files are created at `~/nethackdir`.
The game configuration file is `~/.nethackrc`.
